# SkillCraft Technology Internship - Task 04

## Traffic Accident Data Analysis

### Objective
Analyze traffic accident data to identify patterns related to:
- Time of Day
- Weather Conditions
- Accident Severity
- Visibility
- Contributing Factors

### Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

### Key Findings
- Accident frequency varies by hour.
- Weather affects accident occurrence.
- Visibility impacts road safety.
- Severity changes under different conditions.

### Future Scope
- Accident Prediction using Machine Learning
- Real-time Traffic Monitoring
- Smart Traffic Management
